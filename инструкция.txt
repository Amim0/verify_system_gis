запуск полного прогона осуществляется через main.py
-----------------------------------------------------------------
Сразу после запуска идет приглашение к вводу:
Введите имя модели - используется как основной идентификатор, под этим именем сохраняются результаты обучения, экспортируются версии и проходят проверки.
Режим экспорта (clean/corrupt).
 Определяет тип экспорта:
    clean — сохранение модели без изменений
    corrupt — экспорт с внесением искажений в веса
Порог воспроизводимости (0–100) - числовой критерий допустимого отклонения поведения модели при проверке. Используется на этапе верификации для принятия решения о стабильности модели
---------------------------------------------------------------------
стенд для обучения, экспорта и проверки простой посимвольной модели генерации текста на основе LSTM.

обучение модели на тексте(train_text_gen.py)
сохранение обученной модели
экспорт модели (чистый / с искажением весов) (export.py)
генерацию эталонных примеров (reference.py)
проверку воспроизводимости поведения и контроль целостности файла модели (verify.py)
---------------------------------------------------------------------
project/
 ├── data/
 │    └── text.txt             # текст для обучения
 ├── art/
 │    ├── <name>.pickle        # обученная модель
 │    ├── <name>_exported.pickle # экспорт-я модель
 │    ├── <name>_ref.pickle    # эталон 
 │    └── report_<name>.txt    # отчёт проверки
 ├── text_gen.py               # модель 
 ├── train_text_gen.py         # обучение
 ├── reference.py              # генерация эталона
 ├── export.py                 # экспорт модели
 ├── verify.py                 # проверка модели
 └── main.py                   # запуск 
--------------------------------------------------------------------
